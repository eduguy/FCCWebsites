Introduction

Java is one of the most popular languages in the world. It's relatively easy to learn and has many practical applications.

How To Start

Anyone can learn Java. To start, you need three things:

  • Internet Connection
  • A moderately fast computer
  • Some program to write your code in

There are many popular programs that you can use to write your code in. Personally, I prefer using either Eclipse or IntelliJ. You can quite easily find setup guides for these programs online.

Hello World

To get started, try printing out Hello World in your console by typing in the code below, then running it!

System.out.println("Hello World")

Declaring Variables

In Java, you need to do two things to be able to use a variable:

  • Declare the variable
  • Instantiate the variable

Declaring a variable looks like this:

int x;

You can do this with any of java's primitve types. Instantiating a variable looks like this:

x=5;

You can combine them into one statement as shown below:

int x = 5;

Variable Scope

As in most programming languages, variables have scope.

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the current document. When you declare a variable within a function, it is called a local variable, because it is available only within that function. For example, take a look at this code below: **This information taken from FCC example website**

int x = 5; if (x == 5) { System.out.println("Hello world"); }

This will print out the statement since the variable x is in global scope.
